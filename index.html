/ Habilidades

Lista de Habilidades

HTML & CCS

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas da web. Com ele, definimos elementos como títulos, parágrafos, imagens, links e botões. CSS (Cascading Style Sheets) é responsável pela aparência visual desses elementos. Ele define cores, tamanhos, espaçamentos, fontes e efeitos visuais. Juntos, HTML e CSS criam o layout e o design de sites, formando a base do desenvolvimento front-end.

Javascript

JavaScript é uma linguagem de programação usada para tornar páginas da web interativas. Com ele, podemos criar animações, validar formulários, mostrar mensagens, controlar eventos (como cliques ou teclas) e modificar o conteúdo da página em tempo real.

ReactJS

React.js é uma biblioteca JavaScript usada para construir interfaces de usuário (UI), especialmente em aplicações web. Criada pelo Facebook, ela facilita o desenvolvimento de páginas dinâmicas, rápidas e organizadas, permitindo dividir a interface em componentes reutilizáveis. React usa um conceito chamado DOM virtual, que melhora o desempenho ao atualizar só o que mudou na tela. É muito usada em sites modernos e aplicações interativas.
